# kivy-tasks-client

### Install commands for kivy on Ubuntu:

```bash
sudo add-apt-repository ppa:kivy-team/kivy
sudo apt-get update
sudo apt-get install python3-kivy python-kivy-examples
sudo apt-get install -y \
    python3-pip \
    build-essential \
    mercurial \
    git \
    python3 \
    python3-dev \
    ffmpeg \
    libsdl-image1.2-dev \
    libsdl-mixer1.2-dev \
    libsdl-ttf2.0-dev \
    libsmpeg-dev \
    libsdl1.2-dev \
    libportmidi-dev \
    libswscale-dev \
    libavformat-dev \
    libavcodec-dev \
    zlib1g-dev

pip3 install numpy
pip3 install Cython==0.21.2
pip3 install hg+http://bitbucket.org/pygame/pygame
pip3 install git+https://github.com/kivy/buildozer.git@master
pip3 install git+https://github.com/kivy/plyer.git@master
pip3 install -U pygments docutils
pip3 install kivy
```
